Operating System

Windows File Saving Experience

A usability improvement focused on simplifying the file-saving experience on Windows by aligning the system with the user's preference for local file storage instead of cloud storage.

Problem

The user found the default cloud storage workflow confusing and unnecessary, requiring extra steps to save files and making the process less efficient.

Goal

Reduce the number of steps required to save files by making local storage the default option and creating a smoother, more intuitive experience.

Design Solution

Removed the cloud storage integration and configured Windows to save files to the default local location. This streamlined the workflow, eliminated unnecessary prompts, and made saving files faster and more predictable.

My Role

Observed the user's workflow, identified usability pain points, implemented the system changes, tested the updated experience, and ensured the new workflow matched the user's needs while improving overall efficiency.
